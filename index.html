About me

Hi, I'm Jan Grdanjski. I'm currently attending Technical School Ruđer Bošković in Zagreb, Croatia, as a Computer Science Technician. I spend a lot of my free time doing programming. I'm mostly interested in desktop development, systems programming, backends and most under-the-hood stuff. I started out back in summer of 2018 with Python and some toy game dev. Now I can do Python, C, JavaScript (with and without Node.js), Assembly languages, HTML, CSS (usually Sass) and use frameworks like Express.js, Vue, Django and React. I'm familiar with both SQL and NoSQL databases, usually preferring NoSQL. My go-to for SQL databases is SQLite and for NoSQL databases it's MongoDB. I'm always interested to collab on projects so just hit me up.
